##  Trigonometric Function Plotter

A desktop application built with **Python** and **PyQt5** that allows users to visualize trigonometric functions interactively.

---

### Features
- Plot **Sine, Cosine, Tangent, and Cotangent** functions
- Input angle in **degrees**
- Dynamic graph rendering using **Matplotlib**
- Highlight selected angle on the graph with guide lines
- Display exact function value at the selected angle
- Handle undefined cases (e.g. tan(90Â°), cot(0Â°)) with clear messages
- Smooth and responsive UI designed with **Qt Designer**


#### 2. Run the Application

```bash
python TrigPlotter.py
```
